Приветствую тебя!

Это инструкция по использованию простого чат-бота, который будет выполнять математические операции и выводить результат в окно чата.

КАК ЗАПУСТИТЬ БОТА:
1) Вы просто можете перейти по ссылке — эта ссылка ведет на страницу с результатом.
2) Вы можете скачать файлы. Они скачаются ZIP-файлом, так как GitHub за вас их архивирует. Вы их разархивируете и запускаете файл index.html. Но это не нужно, если не хотите устанавливать бота себе на компьютер.

ЧТО МОЖЕТ БОТ: 

Команды:
1) /start — начало общения с чат ботом.
2) /name: Имя - Запоминает имя
3) /stop - Конец общения
4) /numbers: - Запоминает введенные целые числа
5) /set: - Позваляет хранить переменную

Форматы чисел, которые понимает бот:
1) Запятые - Пример - 7, 9, 12 или 7,9,12
2) Пробелы - Пример - 7 9 12 или 7   9  12
3) Точка с запятой - Пример - 7;9;12 или 7; 9; 12
4) Диапазоны - Пример - 10..15
5) Смешанный - Пример - 5, 10...15, 20

Математические операции, которые умеет выполнять бот:
1) \+ - Сумма
2) \- - Разность (первое минус остальные)
3) \* - Произведение
4) / - Деление (первые на остальные)
5) min - Минимальное
6) max - Максимальное
7) avg - Среднее арифметическое
8) gcd - Наибольший общий делитель

Цепные вычисления и переменные:
1) Бот запоминает последний результат как переменную ans
2) Бот умеет создавать и хранить переменные через команду /set
