Приветствую тебя!
Это Инструкция по использованию простого Чат-бот, который будет выполнять математические операции и выводить результат в окно чата.

КАК ЗАПУСТИТЬ БОТА:
1) Вы просто можете перейти по ссылке - Эта ссылка ведет на Page результат!
2) Вы можете скачать Файлы! Они скачаются Zip файлом так как GitHub за вас их архивирует! Вы их разархивируете и запускаете файл index.html Но это не нужно Если не хотетите устанавливать бота себе на компьютере

ЧТО МОЖЕТ БОТ:
Команды:
 
/start — начало общения с чат ботом.
/name: Имя 
/stop 
/numbers: 

Форматы чисел, которые понимает бот:
Запятые - Пример - 7, 9, 12 или 7,9,12
Пробелы - Пример - 7 9 12 или 7   9  12
Точка с запятой - Пример - 7;9;12 или 7; 9; 12
Диапазоны - Пример - 10..15
Смешанный - Пример - 5, 10...15, 20

Математические операции, которые умеет выполнять бот:
+ - Сумма
- - Разность (первое минус остальные)
* - Произведение
/ - Деление (первые на остальные)
min - Минимальное
max - Максимальное
avg - Среднее арифметическое
gcd - Наибольший общий делитель

Цепные вычисления и переменные:
Бот запоминает последний результат как переменную ans
Бот умеет создавать и хранить переменные через команду /set:
